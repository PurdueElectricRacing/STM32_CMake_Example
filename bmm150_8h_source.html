23// Suspend mode is the default power mode of BMM150 after the chip is powered
24// The device can switch into Active mode from Sleep mode by setting OpMode bits (register 0x4C).
25
26/*
27DATAX_LSB (0x42) contains 5-bit LSB part [4:0] of the 13 bit output data of the X-channel.
28DATAX_MSB (0x43) contains 8-bit MSB part [12:5] of the 13 bit output data of the X-channel.
29DATAY_LSB (0x44) contains 5-bit LSB part [4:0] of the 13 bit output data of the Y-channel.
30DATAY_MSB (0x45) contains 8-bit MSB part [12:5] of the 13 bit output data of the Y-channel.
31The width of the Z-axis magnetic field data is 15 bit word stored in two’s complement.
32DATAZ_LSB (0x46) contains 7-bit LSB part [6:0] of the 15 bit output data of the Z-channel.
33DATAZ_MSB (0x47) contains 8-bit MSB part [14:7] of the 15 bit output data of the Z-channel.
34*/
35// All signed register values are in two´s complement representation.
36
37/*
38The “Data ready status” bit (register 0x48 bit0) is set “1” when the data registers have been
39updated but the data was not yet read out over digital interface. Data ready is cleared (set “0”)
40directly after completed read out of any of the data registers and subsequent stop condition (I²C)
41or lifting of CSB (SPI).
42*/
